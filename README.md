# ShopHub E-Commerce System

ShopHub is a simple e-commerce system designed to demonstrate
object-oriented programming (OOP) concepts and basic system design
using Java and UML diagrams.

This repository contains **design documents** and **Java OOP examples**
as part of the assigned tasks.
